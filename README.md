## Description

Using JS,HTML,SCSS, BEM technologies

Adaptive layout with resolutions:

- Large screens 2560px
- Full HD 1920px
- The design 1600px
- Notebook 1280px
- Tablet 1024
- Mobile (> 320px)

## Local development

### Dependencies
* Node v12.16.3 and higher
* NPM v6.14.4 and higher

### Installing
* Fork and clone this repository
* Run npm install in your terminal
* Run npm start
